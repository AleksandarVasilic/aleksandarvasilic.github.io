Notes

  • Each spreadsheet file is called a “Workbook”
  • A Workbook can contain many Worksheets/Spreadsheet

We can think of file as a book and pages of that book are Spreadsheet

A Worksheet is made up of Cells, and the Cells are organized into Columns and Rows. Number of columns and rows depend on the version of the excel. Latest versions have milions of cells that we can use. Each cell has a unique reference. It takes a letter of the column and the number of the row. Every selected cell has a thick border.

Excel has a ability to create a macros. This is very usefull feacture, we can record what we want to do and then we can repeat that process over and over.

QAT (Quick Access Toolbar) - is a costumizable toolbar that contains a set of commands that are independednt of the tab on the ribbon that is currently displayed. (save, undo etc…)

Absolute reference is where a specified cell or range stays referenced. $ is used to achive an Absoulte Reference
