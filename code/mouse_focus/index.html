Window focus follows the cursor

If you are coming from GNU/Linux background, you might notice that Desktop Environment from Windows OS doesn’t have the ease of focus that comes from hovering over open programs window.

Now, this little annoyance can be corrected by running a CMD script.

The following code:

$signature = @"
[DllImport("user32.dll")]
public static extern bool SystemParametersInfo(int uAction, int uParam, ref 
int lpvParam, int flags );
"@

$systemParamInfo = Add-Type -memberDefinition  $signature -Name SloppyFocusMouse -passThru

[Int32]$newVal = 1
$systemParamInfo::SystemParametersInfo(0x1001, 0, [REF]$newVal, 2)
