Problems with terminal keystrokes while using SSH

Sometimes some terminal emulators don’t respond well to the established ssh connection. Backspace and some other keystrokes simply do not work at all.

This problem does not occur on all terminal emulators, but on some such as kitty or alacritty and maybe some other terminal emulators that I do not know of, this problem is occurs often. For the record, I think that gnome-terminal and xfce4-terminal don’t have this problem.

A command that fixes this issue is:

export TERM=vt100
