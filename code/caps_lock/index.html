Caps Lock as Esc and Mod key

Setting up Caps Lock as an Esc and Mod key

This set of commands will change the default keyboard layout by swapping Caps Lock key with the Esc key.

Now typically this can be done by adding one or two lines of code in your .bashrc file or the i3 config file, but what I urge you to do is that you put this in the .profile in your /home/# folder.

Not only do these commands swap the Esc key and Caps Lock key, but also if we hold the Caps Lock key for long enough, the mentioned key starts behaving like a Mod key.

This comes extremely useful if you are using tiling window managers ( such as i3wm, dwm, xmonad, etc… ) in combination with vi text editors ( vi, vim, neovim… ).

For example, if we type something in vim and we want to exit the INSERT mode and go back to NORMAL mode, what we need to do is just slide our pinky finger to the left of “A” key and get our job done. The same goes for changing the current window in the TWM, we don’t have to awkwardly hold our Mod key which is in most cases the Windows key.

Set of commands that you put in .profile

#increase key speed via a rate change
xset r rate 300 50
#Map the caps lock to super...
setxkbmap -option caps:super
#When is pressed only once, treat it as escape
killall xcape 2>/dev/null ; xcape -e 'Super_L=Escape'
#Map the neby button to right super as well.
xmodmap -e 'keycode 135 = Super_R'
#Turn off the caps lock if on since there is no longer a key for it
xset -q | grep "Caps Lock:\s*on" && xdotool key Caps_Lock
