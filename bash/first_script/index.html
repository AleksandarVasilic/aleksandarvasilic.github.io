First Script

Creating our first script

What we need to do is create file with extension of .sh. Ex.:

touch hello.sh

We can open that file by using a text editor of your choice and then adding the following to it:

echo 'Hello Word!' 
# echo is a command that outputs strings that are passed as arguments

All what is left to do is to run our newly created script:

bash hello.sh
