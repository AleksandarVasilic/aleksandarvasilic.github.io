Bash Shell Scripting

Bourne Again SHell aka Bash is the GNU Project created by Brian Fox as a replacement for the Bourne shell Today Bash is the default shell in almost all GNU/Linux distributions as well it is the default shell in older macOS systems.

As said before some systems do not come with bash as the default shell, so if we want to know if we have bash installed on our system, we should run the following command:

bash --version

We should focus so much on the versions of the bash shell since the syntax does not change that much.

What is shell

Shell is a program that can interpret and execute commands.
