~ sign in the shell prompt tells us that we are currently in the users home directory.

All commands in GNU/Linux are case sensitive so we need to keep an eye on what and how we are typing something. Ex. pwd and PWD are not the same, pwd command will print the name of the current/working directory, while PWD won’t execute because that command does not exist.

Must know commands

  • pwd - prints the name of the current/working directory
  • ls - list directory contents
  • cd - is command for changing directories. It takes an argument as the path that we want to change directory to. Paths can be absolute or relative. Ex. cd /etc/ Printing all the contents of the directory. ls -l gives us more information about the content of the dir. It list the content as the list but also it give us some additional informations.
lrwxrwxrwx   1 root root     7 Jul 20 17:27 bin -> usr/bin
drwxr-xr-x   4 root root  4096 Jul 27 12:53 boot
drwxr-xr-x   2 root root  4096 Jul 20 17:30 cdrom

First column tells us the priveleges and file types of the content.

Ex. drwxr-xr-x first character of this sequence tells us that if the item is directory, file or link. d - directory, - - file, l - link

Second column is the indicator for how many memory locations/blocks directorium/file has.

Third column is reserved for telling us which name of the given file is owner of that file.

Fourth column tells us the group owner of the file.

In GNU/Linux systems besides the name of the users there are name of the groups which certain user can belong to. One user can be part of the one or more groups. Ex. if our user belongs to the root group, it could access the root files.

Fifth column

tells us how much space in bytes given file types contain. Note! 4096 donen’t mean that the given directory with all its content contain 4 Kilobytes, just the given directory without its content contains contain 4K, but the content inside of it are independent files and directories that could contain other files and directories

Sixth column

Tells us the date of last change that has occured .

Seventh column

Tells us the name of the given file/directory.

Now we will change our directory to /etc/
