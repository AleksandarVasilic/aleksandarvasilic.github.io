First command

Now we will run our first command. simply enter pwd in our command line and hit enter.

user@name:~$ pwd
/home/user

As we can see, pwd prints out the name of working directory hence the name pwd (Print Working Directory).

NOTE!

All commands in GNU/Linux are case sensitive so we need to keep an eye on what and how we are typing something. Ex. pwd and PWD are not the same, pwd command will print the name of the current/working directory, while PWD won’t execute because that command does not exist.

–help

Alternatevely to man we can use –help command. It behaves simerarly to the man command but it gives us much shorter answer for using certain command.

cd

Next very important command is that for navigating our file system. That command is cd which stands for change directory. It takes an argument as the path that we want to change directory to. Paths can be absolute or relative.

Now we will change our working directory from /home/user/ directory to / ( also known as root directory ). Simply type:

user@name:~$ cd / # Hit Enter/Return button
user@name:/$      # As wee can see our "~" symbol became "/"
user@name:/$ pwd  # Just to make sure that we are in our root directory
/                 # Returned path
