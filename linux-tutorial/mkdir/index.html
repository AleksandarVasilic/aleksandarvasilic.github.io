Mkdir

Go back to your home folder.

user@name:/$ cd ~
user@name:~$

Creating our first directory

Command for making directories is mkdir which has to have a name of folder as parameter for it to create directory on specified path. Note! Look out for capitalized letters, since it is not the same naming directory project or Project. I will demonstrate this bellow

user@name:~$ mkdir Project
user@name:~$ cd Project
# !If we named our folder with capital letter we need to enter a specific name, cd project won't work!
user@name:/Project$ mkdir documentation
user@name:/Project$ mkdir program
user@name:/Project$ ls -l
total 8
drwxrwxr-x 2 user user 4096 Jul 29 14:41 documentation
drwxrwxr-x 2 user user 4096 Jul 29 14:41 program

We can create multiple folders at once

user@name:/Project$ cd program
user@name:/Project/program$ mkdir dir1 dir2 dir3
# we need to space out wanted directories 
user@name:/Project/program$ ls -l
total 12
drwxrwxr-x 2 user user 4096 Jul 29 14:44 dir1
drwxrwxr-x 2 user user 4096 Jul 29 14:44 dir2
drwxrwxr-x 2 user user 4096 Jul 29 14:44 dir3

touch

touch command behaves the same as the mkdir command but instead of making directories it makes files.

user@name:/Project/program$ touch file1 file2 file3
user@name:/Project$ ls -ltotal 12
drwxrwxr-x 2 user user 4096 Jul 29 14:44 dir1
drwxrwxr-x 2 user user 4096 Jul 29 14:44 dir2
drwxrwxr-x 2 user user 4096 Jul 29 14:44 dir3
-rw-rw-r-- 1 user user    0 Jul 29 14:47 file1
-rw-rw-r-- 1 user user    0 Jul 29 14:47 file2
-rw-rw-r-- 1 user user    0 Jul 29 14:47 file3
